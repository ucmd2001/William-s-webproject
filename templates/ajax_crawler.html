Ajax標題爬蟲-for Medium

這次的爬蟲是使用ajax技術的網站，存取標題都包含在另外的Javascript程式內，而不是單純一整包包在Html裡面。
這樣的好處是，每次送出新請求時可以不用整個Html都要重新跑過一次，在使用過程中如果需要做其他事情(登入、登出、 或是去其他網頁)也能不等網頁回應完之後才能行動，而是可以先去點他預設好的Html框架內的功能，提高使用者體驗。
由於每個Ajax網站放置檔案的位置都不相同，這邊我們使用了Medium網站作為示範。
只要按下按鈕，爬蟲就會自動幫你抓出今日Medium的所有標題囉!


Medium 網址: https://medium.com/
一樣可以自己去驗證看看是不是正確的標題喔! (另外為了防止有人濫用，限制一秒後才會顯示結果。)

查詢結果是:
{% if result != undefind %} {% for key,value in result.items()%} {% endfor %}
{{value}}
{% endif %}
